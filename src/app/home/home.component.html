The button at the bottom sends the user to Auth0 to sign in. The token generated during the sign-in process is specific to an environment. This environment can be manipulated by changing the following properties:

{

clientID: ,

domain: ,

audience: ,

scope: ,

}

To share a predefine set of properties, we need a link that contains the following query parameters: clientID, domain, audience, and scope: Auth0 Sample App
